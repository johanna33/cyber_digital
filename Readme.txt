Cyber J Digital POS (point of sales software)

A point of sale desktop application that handles sales, inventory, purchasing, customers, vendors, employees, billing, financial information, reports, and other miscellaneous operations customizable for the business. The system can configure specific stores set for the business with their specific departments allowing more control and organization of the company.

Requirements:
Windows xp or 7
MySQL Server 4.1
MyODBC windows drivers 3.51
BDE Admin 5.01
